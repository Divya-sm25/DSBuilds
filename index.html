FitSync – Fitness Tracker Web App

Tech Stack: The website is built using HTML, CSS, PHP, and MySQL. It is styled with custom CSS and uses PHP for server-side scripting and session management. MySQL is used for storing user data and activity logs. The interface includes dynamic elements rendered based on user input and database queries.

Outcomes: This Fitness Tracker allows users to register and log in securely, track their running and weightlifting activities, log food intake, and calculate calories burned. Users can view personalized activity history, including calories burned, distances run, and weights lifted. The site encourages healthy living by offering a list of nutritious food items along with calorie details.

Veritas Vogue – Fashion Admin Dashboard

Tech Stack- HTML, CSS, JavaScript, PHP, and MySQL.
Veritas Vogue is a modern admin dashboard built for a fashion brand to manage products, customers, and analytics in real time. The interface is responsive, data-driven, and designed with user experience in mind. Responsive and aesthetic UI with black-themed modern design.

Dynamic product listing with category-wise filtering (e.g., Men, Women, Wedding).

"Add to Cart" and Like (Heart) button functionality with UI feedback.

Navigation between Home, Collections, Products, and Contact pages.

Integrated PHP backend to fetch products from MySQL based on category.

URL-based filtering (e.g., products.html?category=mens_wear).

TrackExpiry – Inventory Expiry Tracker

Tech Stack:

• Frontend: HTML, CSS, JavaScript (Dashboard Interface)
• Backend: Python (Logic & Automation)
• Database: Firebase Realtime Database / Firestore
• QR Code: Python QR Code Library (e.g., qrcode)
• Notification System: Firebase Cloud Messaging (FCM)
• Data Visualization: Matplotlib, Pandas
• Hosting: Firebase Hosting (optional for deployment)

Project Outcomes:

• Reduced expired product waste by up to 40% through timely alerts
• Automated daily expiry checks without manual intervention
• Enabled quick product lookup via QR code scanning
• Improved stock turnover rate by applying shelf life-based discounts
• Provided real-time inventory visibility and expiry analytics via dashboard
• Enhanced decision-making through expiry trend reports and charts

Expense Reimbursement Automation – UIpath Workflow

Tech Stack: UiPath Studio, UiPath Orchestrator (Triggers), Excel, PDF Processing, SMTP Email Integration, Windows File System

Outcomes:

✓ Automated extraction of expense data from submitted PDF receipts using UiPath’s document understanding capabilities.

✓ Validated each reimbursement request against predefined business rules (e.g., duplicate check, missing info, limit thresholds).

✓ Stored extracted data in structured Excel sheets for record-keeping and reporting.

✓ Sent automated email updates to employees based on the approval status via SMTP integration.

✓ Enabled manager-level review and approval directly from the automation workflow without manual intervention.

✓ Used UiPath Orchestrator triggers to schedule and execute bots automatically at specific intervals or upon new receipt uploads.

✓ Ensured Finance department was notified and updated the status of completed reimbursements automatically.

✓ Improved transparency and auditability of expense claims throughout the workflow.

✓ Minimized manual errors, increased process efficiency, and ensured timely processing of employee claims.
